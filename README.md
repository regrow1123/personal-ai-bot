# Personal AI Bot

텔레그램 개인 AI 봇 + 멀티 프로바이더 동적 관리

LangChain/LangGraph 학습 프로젝트

## 학습 단계

- [ ] 1단계: 환경 세팅
- [ ] 2단계: 단일 LLM 챗봇
- [ ] 3단계: LangGraph 에이전트 + 메모리
- [ ] 4단계: 커스텀 도구 추가
- [ ] 5단계: 텔레그램 봇 연동
- [ ] 6단계: ModelManager 구현
- [ ] 7단계: 동적 전환 + 자동 로테이션
- [ ] 8단계: 고도화 (RAG, MCP, 장기 메모리)

## 기술 스택

- Python 3.x
- LangChain / LangGraph
- python-telegram-bot
- Ollama (로컬 LLM)
- Google Gemini API
- Anthropic Claude API (선택)

## 설치

```bash
pip install -r requirements.txt
```
